# Use asm x86_64 in C
## Description
This project is about how to use asm x86_64 in C.
In this Example we will xor a string with an integer.

## Compile
```bash
make
```

## Usage
```bash
./main Hello World 2
```

The output will be:
```text
Jgnnm"Umpnf
```
